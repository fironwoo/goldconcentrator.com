## 如何在后台正文里插入图片

1. 打开对应页面，比如 Alluvial Gold。
2. 找到正文 Body / Markdown 编辑区域。
3. 上传图片到后台媒体库。
4. 图片路径通常会是 `/images/uploads/文件名.jpg`。
5. 在正文中插入：

```md
![Alluvial gold wash plant process illustration](/images/process/alluvial-gold-flowsheet.svg)
```

或上传真实图片后插入：

```md
![Trommel scrubber prepared for shipment](/images/uploads/trommel-scrubber-shipment.jpg)
```

图片说明要真实。示意图叫 illustration，真实照片才能叫 factory photo, workshop photo, shipment photo, field photo。

## 案例写法模板

安全写法：

```md
## Project challenge

The feed contained sticky clay and mixed gravel sizes. The first design task was to release the gold-bearing fraction before gravity recovery.

## Process route

The proposed route used feeding control, washing, screening, staged gravity recovery and concentrate cleaning.

## Evidence status

This page is a project pattern until approved photos, test records or customer permission are available.
```

不要这样写：

```md
This customer achieved 95% recovery and doubled production.
```

除非你有测试方法、运行条件、客户授权和可公开证据。
