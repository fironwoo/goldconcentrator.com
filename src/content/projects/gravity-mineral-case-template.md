## Select the strongest documented case

Use the non-gold project with the best combination of representative feed information, process reasoning, approved images and outcome records. A smaller well-documented project is more credible than a large project supported only by equipment photographs.

## Explain the separation problem

Show how mineral identity, liberation size, clay, slimes, associated heavy minerals or product specification affected the route.

## Show the mass-flow logic

Document where material was classified, which streams entered roughing, how middlings were handled and what cleaning stage was required. This is more useful than a simple list of machine names.

## Publish only supported results

If concentrate grade or recovery is available, state the sampling period and test or operating conditions. If it is not available, focus on the verified design and delivery scope.
